# 更好的游戏列表

通过 RELAY-CN 自有资源，在保障 RELAY-CN 稳定运行的同时，面向大众提供反广告游戏列表。  
兼容原版《铁锈战争》API 逻辑，在内部审核环节引入 API 与 IronCore 启发式扫描机制，并实现自动化数据流：

原理为 :
```
- 每 30 秒拉取一次上游 List 数据  
- 每次拉取后调用 AI 审核引擎进行判断  
- 依据审核结果过滤违规条目  
- 对新出现的房间额外执行启发式扫描，确保实时防护  
```

AI 审核词库将公开并持续迭代优化。
